Introduction

Welcome to the Admin User Manual for the Sri Lanka Stay & Explore website. This guide will help you navigate and utilize the features of the admin panel to effectively manage your website's content, bookings, and settings.

Dashboard

The Dashboard provides an overview of your website's activity.

  • Total Visits: Displays the total number of visits to your website.
  • Bookings: Shows the total number of bookings made.
  • Average Rating: Displays the average user rating from feedback.
  • Popular Routes: Lists the most frequently booked transport routes.
  • Recent Feedback: Shows the latest feedback submitted by users.

Charts are provided for a visual representation of visits, bookings, and ratings.

Manage Pictures

This section allows you to upload, organize, and manage all images used on your website, including those in the gallery and carousel.

Uploading a New Picture

  1. Click the " Upload Picture" button.
  2. In the modal that appears, click " Select Image" to choose an image file from your computer. A preview will be shown.
  3. Enter an "Image Name" (descriptive).
  4. Select a "Category" (Scenery, Wildlife, Culture, Food, Beach).
  5. Optionally, add a "Description" for the image.
  6. Click " Upload Image".

Organizing Pictures

  • Click the " Organize Folders" button (if functionality is available for folder management).
  • Use the "Filter by Category" dropdown to view images from specific categories.
  • The picture grid displays all uploaded images. You can typically click on an image to view details or delete it.

Image Carousel

Manage the images that appear in the main image carousel on your website's homepage or other designated areas.

Adding Images to Carousel

  1. Click the " Add Image to Carousel" button.
  2. A modal will appear showing available images from your "Manage Pictures" section.
  3. Select the image(s) you want to add.
  4. Click "Add Selected".

Managing Carousel Images

  • Reordering: Drag and drop images within the "Current Carousel Images" container to change their display order.
  • Click " Save Order" to save any changes to the image sequence.
  • Images in the carousel can usually be removed by clicking a delete icon associated with them.

Content Management

Manage various types of content on your website, such as articles, news, videos, and recommended links.

Use the tabs (Articles & News, Videos, Recommended Links) to switch between content types.

Managing Articles & News

  1. Click the " Add New Article" button.
  2. Fill in the article details in the form/modal that appears (e.g., Title, Content, Category, Featured Image).
  3. Save the article.
  4. Existing articles will be listed, allowing you to edit or delete them.

Managing Videos

  1. Click the " Add New Video" button.
  2. Provide video details, such as a title, description, and the video URL (e.g., YouTube link) or upload a video file.
  3. Save the video.
  4. Existing videos can be edited or deleted from the list.

Managing Recommended Links

  1. Click the " Add New Link" button.
  2. Enter the link's title, URL, and a brief description.
  3. Save the link.
  4. Manage existing links by editing or deleting them.

Accommodations Management

Manage hotel and room information displayed in the "Luxurious Accommodations" section of your website.

Use the tabs (Hotels, Rooms) to switch between managing hotel entries and individual room types.

Managing Hotels

  1. Click the " Add New Hotel" button.
  2. In the modal:
    • Enter "Hotel Name".
    • Specify "Location".
    • Input "Price Range" (e.g., $100-200/night).
    • Select a "Rating" (1-5 Stars).
    • Write a "Description".
    • Upload a "Hotel Image".
    • List "Amenities" (comma-separated, e.g., WiFi, Pool, Restaurant).
    • Optionally, add the "Website URL".
  3. Click "Save Hotel".
  4. Existing hotels can be edited or deleted from the grid view.

Managing Rooms

  1. Click the " Add New Room" button.
  2. In the modal:
    • Enter "Room Name" (e.g., Ocean View Suite).
    • Write a "Description".
    • Input "Price per Night".
    • Specify "Room Size (mÂ²)".
    • Select "Bed Type" (King, Queen, Twin, Double).
    • Indicate if "WiFi" is available (Yes/No).
    • Upload a "Room Image".
  3. Click " Save Room".
  4. Existing rooms can be edited or deleted from the grid view.

Order Management

View and manage all customer orders for transport services.

Filtering and Searching Orders

  • Use the "Search orders..." input field to find specific orders.
  • Click " Clear All Orders" to permanently delete all order data (use with extreme caution).
  • Use the "Filter by User" dropdown to view orders from a specific customer.
  • Use the tabs (All Orders, Pending, Confirmed, Completed, Cancelled) to filter orders by their status.

Viewing Order Details

  • Orders are listed in a table with columns: Order ID, Customer, Date, From, To, Amount, Status, and Actions.
  • Click on an action button (e.g., "View Details" or an icon) in the "Actions" column to open the "Order Details" modal.

Managing Order Status

In the "Order Details" modal:

  • Mark as Pending: Sets the order status to Pending.
  • Confirm Order: Confirms the order.
  • Mark as Completed: Marks the order as completed.
  • Cancel Order: Cancels the order.
  • Click "Close" to exit the modal.

Transport Settings

Configure the pricing and parameters for your transport services.

Base Price and Distance Calculation

  • Base Fare (USD): The starting fare for any journey.
  • Rate per Kilometer (USD): The cost added for each kilometer of travel.

Time-based Multipliers

These multipliers adjust the fare based on the time of travel.

  • Rush Hour Multiplier (6-9 AM, 4-7 PM): Factor by which the fare is multiplied during peak hours.
  • Night Time Multiplier (10 PM - 6 AM): Factor for late-night travel.
  • Weekend Multiplier: Factor for travel on weekends.

Vehicle Types

Set rate multipliers for different vehicle categories.

  • Sedan Rate Multiplier
  • SUV Rate Multiplier
  • Van Rate Multiplier
  • Luxury Vehicle Rate Multiplier

Click "Save Transport Settings" to apply any changes.

Settings

Manage your admin account settings.

  • Admin Username: View or change your admin username.
  • Change Password: Enter a new password in this field.
  • Confirm Password: Re-enter the new password to confirm.

Click "Save Settings" to update your account details.

Logout

Click the " Logout" link in the sidebar to securely log out of the admin panel.
