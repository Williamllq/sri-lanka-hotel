Introduction

Welcome to the Sri Lanka Stay & Explore Admin Manual. This comprehensive guide is designed to help administrators effectively manage the website, its content, and user interactions.

As an administrator, you have access to a range of powerful tools and features that allow you to:

  • Monitor website performance and user activity
  • Manage image content across the site
  • Create and edit articles, videos, and recommended links
  • Manage accommodation listings
  • Process and track customer orders
  • Configure transport pricing and settings
  • Adjust system settings and security options

This manual provides detailed instructions for using each feature of the admin dashboard to ensure smooth operation of the Sri Lanka Stay & Explore website.

Note: Admin access should be limited to authorized personnel only. Never share your login credentials with others.

Admin Dashboard

The admin dashboard provides a comprehensive overview of website performance and key metrics.

Accessing the Dashboard

  1. Navigate to https://sri-lanka-stay-explore.netlify.app/admin-login.html
  2. Enter your admin credentials
  3. Upon successful login, you'll be directed to the admin dashboard

Dashboard Components

The dashboard contains the following key metrics and components:

  • Total Visits: Displays the number of visitors to the website
  • Bookings: Shows the total number of bookings made
  • Average Rating: Displays the average customer rating
  • Popular Routes: Lists the most frequently booked transportation routes
  • Recent Feedback: Shows the latest customer feedback submissions

Navigation

The admin dashboard uses a sidebar navigation system:

  • Click on any sidebar item to navigate to the corresponding section
  • The active section will be highlighted in the sidebar
  • Use the hamburger menu icon (☰) in the top-left corner to toggle the sidebar visibility on mobile devices

Tip: You can quickly return to the front-end website by clicking "View Website" in the sidebar footer.

Manage Pictures

The Pictures Management section allows you to upload, organize, and manage images used throughout the website.

Uploading Pictures

To upload new pictures to the website:

  1. Navigate to "Manage Pictures" in the sidebar
  2. Click the "Upload Picture" button
  3. In the upload modal:
    • Click "Select Image" to choose an image file from your computer
    • Enter a descriptive name for the image
    • Select the appropriate category (Scenery, Wildlife, Culture, Food, Beach)
    • Add a detailed description that will be displayed with the image
  4. Click "Upload Image" to complete the process

Note: For optimal performance, images should be in JPG, PNG, or WebP format, with a recommended size of 1200×800 pixels and less than 500KB in file size.

Organizing Pictures

To organize and manage existing pictures:

  1. Navigate to "Manage Pictures" in the sidebar
  2. Use the category dropdown to filter images by category
  3. For each image, you can:
    • View: Click on the image to view it in full size
    • Edit: Click the edit icon to modify the image details
    • Delete: Click the trash icon to remove the image
    • Add to Carousel: Click the carousel icon to add the image to the homepage carousel

Warning: Deleting an image is permanent and cannot be undone. Make sure the image is not being used elsewhere on the site before deletion.

Image Carousel

The Image Carousel section allows you to manage the rotating images displayed on the homepage.

Adding Images to the Carousel

  1. Navigate to "Image Carousel" in the sidebar
  2. Click the "Add Image to Carousel" button
  3. In the selection modal, browse the available images
  4. Click on the images you want to add to select them
  5. Click "Add Selected" to add the chosen images to the carousel

Reordering Carousel Images

  1. Navigate to "Image Carousel" in the sidebar
  2. In the "Current Carousel Images" section, drag and drop images to change their order
  3. Click "Save Order" to apply the changes

Removing Images from the Carousel

  1. Navigate to "Image Carousel" in the sidebar
  2. For each image in the carousel, click the "Remove" button (X)
  3. The image will be removed from the carousel but will remain in the image library

Tip: For the best visual appearance, select images with similar aspect ratios and compositions. Landscape orientation works best for the carousel.

Content Management

The Content Management section allows you to create and manage various types of content displayed on the website.

Articles Management

To manage articles and news content:

  1. Navigate to "Content Management" in the sidebar
  2. Select the "Articles & News" tab
  3. To add a new article:
    • Click the "Add New Article" button
    • Fill in the article title, content, and select a featured image
    • Add tags and set a publish date
    • Click "Save Article" to publish
  4. To edit an existing article:
    • Locate the article in the list
    • Click the "Edit" button
    • Make your changes in the edit form
    • Click "Update Article" to save changes

Videos Management

To manage video content:

  1. Navigate to "Content Management" in the sidebar
  2. Select the "Videos" tab
  3. To add a new video:
    • Click the "Add New Video" button
    • Enter the video title and description
    • Paste the YouTube or Vimeo embed code or URL
    • Select a thumbnail image or use the auto-generated one
    • Click "Save Video" to publish
  4. To edit an existing video:
    • Locate the video in the list
    • Click the "Edit" button
    • Make your changes in the edit form
    • Click "Update Video" to save changes

Links Management

To manage recommended links:

  1. Navigate to "Content Management" in the sidebar
  2. Select the "Recommended Links" tab
  3. To add a new link:
    • Click the "Add New Link" button
    • Enter the link title, URL, and description
    • Select an icon and category
    • Click "Save Link" to add it to the site
  4. To edit an existing link:
    • Locate the link in the list
    • Click the "Edit" button
    • Make your changes in the edit form
    • Click "Update Link" to save changes

Accommodations Management

The Accommodations Management section allows you to manage hotels and room listings on the website.

Hotels Management

To manage hotel listings:

  1. Navigate to "Accommodations Management" in the sidebar
  2. Select the "Hotels" tab
  3. To add a new hotel:
    • Click the "Add New Hotel" button
    • Fill in the hotel details: name, location, price range, rating, etc.
    • Upload hotel images
    • Add amenities (comma-separated list)
    • Enter the hotel website URL (if applicable)
    • Click "Save Hotel" to publish
  4. To edit an existing hotel:
    • Locate the hotel in the list
    • Click the "Edit" button
    • Make your changes in the edit form
    • Click "Update Hotel" to save changes

Rooms Management

To manage room listings:

  1. Navigate to "Accommodations Management" in the sidebar
  2. Select the "Rooms" tab
  3. To add a new room:
    • Click the "Add New Room" button
    • Fill in the room details: name, description, price per night, etc.
    • Specify room size and bed type
    • Indicate available amenities
    • Upload room images
    • Click "Save Room" to publish
  4. To edit an existing room:
    • Locate the room in the list
    • Click the "Edit" button
    • Make your changes in the edit form
    • Click "Update Room" to save changes

Tip: High-quality room images significantly increase booking rates. Use professional photography when possible, and include multiple images showing different angles and features of each room.

Order Management

The Order Management section allows you to view, process, and manage customer bookings and orders.

Viewing Orders

To view and filter orders:

  1. Navigate to "Order Management" in the sidebar
  2. By default, all orders are displayed
  3. Use the tab filters to view orders by status:
    • All Orders
    • Pending
    • Confirmed
    • Completed
    • Cancelled
  4. Use the search box to find specific orders by customer name, order ID, or location
  5. Use the user filter dropdown to view orders from specific customers

Processing Orders

To process and update orders:

  1. Locate the order you want to process in the orders table
  2. Click the "View" button to see detailed information
  3. In the order details modal:
    • Review all order information, including customer details, booking information, and payment status
    • Update the order status using the action buttons:
      • "Mark as Pending" - Sets the order status to pending
      • "Confirm Order" - Confirms the booking
      • "Mark as Completed" - Marks the service as delivered
      • "Cancel Order" - Cancels the booking
    • Add internal notes if needed
  4. Click "Close" to exit the details modal

Note: When an order status is changed, the customer will automatically receive an email notification about the status update.

Warning: The "Clear All Orders" button will permanently delete all order data. Use this function with extreme caution and only when absolutely necessary.

Transport Settings

The Transport Settings section allows you to configure pricing and rates for transportation services.

Base Pricing Configuration

To configure base pricing:

  1. Navigate to "Transport Settings" in the sidebar
  2. In the "Base Price and Distance Calculation" section:
    • Set the "Base Fare (USD)" - the minimum fare for any journey
    • Set the "Rate per Kilometer (USD)" - the amount charged per kilometer traveled

Time-based Multipliers

To configure time-based pricing multipliers:

  1. In the "Time-based Multipliers" section:
    • Set the "Rush Hour Multiplier" - applied during peak traffic hours (6-9 AM, 4-7 PM)
    • Set the "Night Time Multiplier" - applied for night journeys (10 PM - 6 AM)
    • Set the "Weekend Multiplier" - applied for weekend bookings

Vehicle Type Rates

To configure vehicle-specific rates:

  1. In the "Vehicle Types" section:
    • Set the "Sedan Rate Multiplier" - for standard sedan vehicles
    • Set the "SUV Rate Multiplier" - for SUV and larger vehicles
    • Set the "Van Rate Multiplier" - for passenger vans
    • Set the "Luxury Vehicle Multiplier" - for premium and luxury vehicles

After making any changes, click "Save Transport Settings" to apply the new rates.

Tip: Price adjustments will affect new bookings only. Existing confirmed bookings will maintain their original pricing.

Fare Calculation Formula

The system calculates fares using the following formula:

Base Fare = Distance (km) × Rate per Kilometer
Time Adjustment = Base Fare × Applicable Time Multiplier
Vehicle Adjustment = Base Fare × Vehicle Type Multiplier
Total Fare = Base Fare + Time Adjustment + Vehicle Adjustment

Admin Settings

The Settings section allows you to manage administrator accounts and system configurations.

Account Management

To update your admin account:

  1. Navigate to "Settings" in the sidebar
  2. In the account management section:
    • Update your username if needed
    • Change your password (requires entering the new password twice for confirmation)
  3. Click "Save Settings" to apply changes

Warning: Choose a strong password that includes a mix of uppercase and lowercase letters, numbers, and special characters. Admin account security is crucial for website protection.

System Configuration

Additional system settings may be available depending on your admin level:

  • Email notification settings
  • Payment gateway configuration
  • Backup and maintenance options
  • API integration settings

Access to these settings may require super-admin privileges.

Troubleshooting

Common issues administrators may encounter and how to resolve them.

Image Upload Issues

If you experience problems with image uploads:

  • Ensure the image file is in a supported format (JPG, PNG, WebP)
  • Check that the file size is less than 5MB
  • Try compressing the image before uploading
  • Clear your browser cache and try again
  • Use a different browser if the issue persists

Dashboard Not Loading Properly

  • Clear your browser cache and cookies
  • Try using a different browser
  • Ensure you have a stable internet connection
  • Check that JavaScript is enabled in your browser
  • Try logging out and logging back in

Data Not Saving

  • Check that all required fields are filled out
  • Look for any error messages that might indicate specific issues
  • Ensure your session hasn't timed out (log out and back in)
  • Try saving smaller amounts of data at a time

Admin Access Issues

  • If you can't log in, try the password reset function
  • Ensure you're using the correct URL for admin access
  • Check if your admin privileges have been changed
  • Contact the system administrator if you continue to have issues

Note: If you encounter persistent technical issues that you cannot resolve, please contact the website developer for assistance.
